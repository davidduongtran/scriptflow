## 🎯 STYLE-TO-CONTENT MATCHING

### Finance Content
| Topic | Recommended Style |

| Investing basics | Isometric 3D |

| Crypto/NFT | Neon Cyberpunk |
| Real estate | Photorealistic or Cinematic |

## ⚡ QUICK SELECTION GUIDE

```
Creating Finance Video?
├── Educational/Tutorial → Whiteboard or Isometric 3D
├── Story-driven → 3D Animation or Cinematic
├── Data/Charts → Isometric 3D
├── Quick Tips → 2D Minimal
└── Crypto/Tech → Neon Cyberpunk

Creating Psychology Video?
├── Mental Health → Anime or Watercolor
├── Relationships → Anime
├── Trauma/Sensitive → Silhouette
├── Research/Studies → Whiteboard
└── Deep Concepts → 2D Minimal
```

## 🔧 TECHNICAL DNA CHEAT SHEET

### For Consistent VEO 3 Output

**3D Animation:**
```
ray-traced PBR, subsurface scattering, global illumination, 24fps, 4K
```

**Anime:**
```
cel-shaded, expressive eyes, soft pastel palette, 24fps on 2s, manga-influenced
```

**Whiteboard:**
```
black marker on white, hand-drawing reveal, red/blue accents, variable fps
```

**Watercolor:**
```
bleeding wash edges, paper texture, soft pastels, dreamy atmosphere, 24fps
```

**Isometric 3D:**
```
30-degree projection, clean geometry, no perspective distortion, pastel palette
```
