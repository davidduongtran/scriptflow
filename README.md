## ✨ Features

### 📝 Script Generation Tab
- **Claude AI Integration** - Professional YouTube scripts
- **Multiple Templates** - Editorial, Documentary, Tutorial, VSL, Shorts
- **Clone Mode** - 100% verbatim or 95% creative reproduction
- **AI Suggestions** - Auto-populate fields from analyzer data

### 🎥 VEO Prompts Tab
- **Gemini AI Integration** - Generate Flow VEO 3 prompts
- **Flow Extend Format** - Scene-by-scene with timing, camera, SFX
- **25+ Visual Styles** - Comprehensive style dimensions
- **Batch Generation** - Handle 20+ prompts without errors

### 📊 Assessment Tab
- **Prompt Quality Audit** - Score and improve prompts
- **Auto-Fix Suggestions** - One-click improvements
- **Export Reports** - JSON, Markdown, PDF formats

### ⚙️ Settings Tab
- **API Key Management** - Claude and Gemini
- **Default Configurations** - Templates, lengths, models

## 📦 Installation

1. Open Chrome → `chrome://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `TITAN_MIND` folder
5. Pin to toolbar

## 🚀 Quick Start

### From Scratch
1. **Script Tab** → Enter topic, audience, angle
2. Select template and AI model
3. Click **Generate Script**
4. Click **🎬 → VEO** to transfer

### From TITANSYS SCOPE / TEXT
1. Analyze content in source extension
2. Click **Send to TITANSYS MIND**
3. Data auto-populates
4. Review and **Generate Script**

### VEO Workflow
1. **VEO Prompts Tab** → Paste or transfer script
2. Configure style settings (25+ visual styles)
3. Click **Generate All Prompts**
4. Copy prompts to Google Flow
